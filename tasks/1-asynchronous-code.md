# Асинхронный код

Реализуйте и экспортируйте по умолчанию асинхронную функцию, которая читает данные файла по указанному пути и выводит их в консоль.

Пример:

```js
import print from '../solutions/1-asynchronous-code.js';
print('./myfile');
```
